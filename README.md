This project is a Python classic Magic 8 Ball game:

This project is a Python creation of the Magic 8 Ball game. The program allows a user to ask a question and receive a randomly generated response.

I have implemented the random module to generate different numerical outcomes with the random function corresponding to a range of answers. Additionally, I have also utilised conditional logic with if/Elif/else statements to control the flow of the program.
 This project has strengthened my understanding of basic programming concepts such as variables, print statements, control flow, and user interaction. Moreover, it improved my problem-solving skills by handling edge cases, for example if a user doesn’t provide a name, answer, or both.
